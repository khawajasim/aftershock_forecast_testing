experiment.grid_operations.forecast_aggregation(test_grid, model_grid, stress_value, by_addition=False)[source]

Forecast mapping from one grid “model_grid” to the “test_grid” Input:

test_grid: A list of polygons of test grid model_grid: A list of polygons of forecast grid stress: forecast

Process: If by_addition=True, then add the fraction of stress shared by the common areas.

If by_addition=False, then average stress of all the points touching a cell. (Since, the stress is computed for the points, so I find the latter better for aggregation.)
